# Fantia 一括非公開ツール

Fantia の自分の投稿を **一括で非公開化**、もしくは **再公開** できる Electron デスクトップアプリです。

## 何ができる？

- 自分のすべての投稿を一括で「非公開」に切り替える
- 上記操作で非公開化したものだけを、まとめて「公開」に戻す
  - **元から非公開だった投稿は影響しません** (履歴ベースの再公開)
- 「現在非公開のすべての投稿を公開化」する別ボタンも用意 (元から非公開だったものも含めて公開してしまうので注意)
- 非公開化した投稿の ID 一覧を CSV でダウンロード
- 進捗のリアルタイム表示、途中中断、リクエスト間隔の調整

## ダウンロード

[Releases ページ](../../releases) から OS 向けのインストーラをダウンロードしてください。

- macOS: `Fantia-Unpublish-Tool-*.dmg`
- Windows: `Fantia-Unpublish-Tool-*.exe`

### macOS で「開発元未確認」と出る場合

署名証明書をつけていないため、初回起動時に警告が出ます。以下のどちらかで起動できます:

- Finder で .app を右クリック → 「開く」
- またはターミナルで隔離属性を外す:
  ```sh
  xattr -dr com.apple.quarantine "/Applications/Fantia Unpublish Tool.app"
  ```

### Windows で SmartScreen 警告が出る場合

「詳細情報」→「実行」で起動できます。

## 使い方

1. アプリを起動すると、操作パネル (左) と Fantia ブラウザ (右) の 2 つのウィンドウが開きます
2. 右側のブラウザで Fantia にログイン
   - 別ページに移ってしまった場合は「Fantia のログイン画面を開く」ボタンで戻れます
3. 操作パネルで実行したいボタンを押す:
   - **すべての投稿を非公開にする** — 現在公開中のすべての投稿を非公開化
   - **このツールで非公開にしたものを公開する** — 履歴ベースの安全な再公開
   - **非公開のものを全部公開する** — 現在非公開状態のすべての投稿を公開化 (元から非公開だったものも含む)
4. 完了するとダイアログで通知されます

非公開化した投稿の ID は、OS のユーザーデータディレクトリに保存されます (アプリをアンインストールしても残ります)。
バックアップしておきたい場合は「履歴を CSV ダウンロード」を使ってください。

## 仕様メモ

- セッション/Cookie はアプリ専用パーティション (`persist:fantia-unpublish`) に保存
- 操作はすべて Fantia のマイページ編集画面経由で実行されます
- レート制限考慮のため、1 件ごとに数秒の待機を入れています (UI で調整可)
- 投稿一覧の取得には `?q[status_eq]=1/0` のフィルタ URL を使用 (状態別に絞り込み)

## 開発

```sh
npm install
npm run start      # 開発実行
npm run dist:mac   # macOS .dmg ビルド
npm run dist:win   # Windows .exe ビルド (Wine が必要)
```

## リリースの作り方 (メンテナ向け)

GitHub Actions が `v*` タグの push で発火し、macOS と Windows のランナーで `.dmg` / `.exe` をビルドして Release に添付します。

```sh
# バージョン更新後
git tag v0.1.0
git push origin v0.1.0
```

手動でビルドだけ走らせたい場合は Actions タブから "Release" ワークフローを `workflow_dispatch` で実行できます (成果物は Artifacts に保存)。
