# Fantia 一括非公開ツール

Fantia の自分の投稿をまとめて **非公開** にしたり、 **公開に戻したり** できるデスクトップツールです。

## ダウンロード

[Releases ページ](../../releases) から、自分の OS のファイルを落としてください。

| OS | ファイル |
| --- | --- |
| Windows | `Fantia-Unpublish-Tool-*.exe` |
| macOS | `Fantia-Unpublish-Tool-*.dmg` |

### 起動時に警告が出たら？

このアプリは初回だけ OS が警告を出します。以下の手順で起動できます。

**Windows**: 「WindowsによってPCが保護されました」と出たら → **「詳細情報」** → **「実行」**

**macOS**: 「開発元を確認できません」と出たら → Finder で `.app` を **右クリック → 開く**

あと mac は起動にやたら時間がかかります。(2,3分)


## 使い方

### 1. 起動する

アプリを開くと **左に操作パネル**、 **右に Fantia のブラウザ** が並んで開きます。

### 2. Fantia にログインする

右側のブラウザで Fantia のログイン画面が開きます。ここでメアドを使ってログインしてください。(Google アカウントではログインができないと思います。その場合はメアドログインできるように Fantia から設定してください。)

もし違うページに行ってしまったら、左パネルの **「Fantia のログイン画面を開く」** で戻れます。

### 3. やりたい操作を行う

操作パネルに 3 つのボタンがあります。

* すべての投稿を非公開にする
* このツールで非公開にしたものを公開する  (もともと非公開だったものを公開しないようにするためです) 
* 非公開のものを全部公開する  (もともと非公開だったものも公開されるので注意) 

進捗バーで何件目を処理しているか見られます。完了するとダイアログでお知らせします。

### 4. リクエスト間隔

Fantia への負荷を考慮し、投稿 1 件ごとに数秒待つようになっています。

## Q&A 

### Q. アプリを一度閉じても、「このツールで非公開にしたもの」の記憶は残る？

残ります。 アプリを終了しても次回起動時にちゃんと再公開できます。

### Q. 履歴をバックアップしたい

操作パネルの **「履歴を CSV ダウンロード」** で、非公開にした投稿の ID 一覧を CSV で保存できます。

### Q. 履歴を消したい

**「非公開化履歴をクリア」** で消せます。クリア後はこのアプリから一括公開できなくなるので注意。

### Q. ログインし直したい

右側の Fantia ブラウザでログアウトすれば普通にログアウトできます。
