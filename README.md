# Fantia 一括非公開ツール

Fantia の自分の投稿をまとめて **非公開** にしたり、 **公開に戻したり** できるデスクトップアプリです。

ボタンをポチポチするだけで、何百件の投稿でも自動で順番に切り替えてくれます。

---

## ダウンロード

[Releases ページ](../../releases) から、自分の OS のファイルを落としてください。

| OS | ファイル |
| --- | --- |
| Windows | `Fantia-Unpublish-Tool-*.exe` |
| macOS | `Fantia-Unpublish-Tool-*.dmg` |

### 起動時に警告が出たら？

このアプリは署名証明書をつけていないので、初回だけ OS が警告を出します。中身は安全なので以下の手順で起動できます。

**Windows**: 「WindowsによってPCが保護されました」と出たら → **「詳細情報」** → **「実行」**

**macOS**: 「開発元を確認できません」と出たら → Finder で `.app` を **右クリック → 開く**

それでも開けない時はターミナルで:

```sh
xattr -dr com.apple.quarantine "/Applications/Fantia Unpublish Tool.app"
```

## 使い方

### 1. 起動する

アプリを開くと **左に操作パネル**、 **右に Fantia のブラウザ** が並んで開きます。

### 2. Fantia にログインする

右側のブラウザで普段どおり Fantia にログインしてください。

もし違うページに行ってしまったら、左パネルの **「Fantia のログイン画面を開く」** で戻れます。

### 3. やりたい操作のボタンを押す

操作パネルに 3 つのボタンがあります。

| ボタン | やること |
| --- | --- |
| **すべての投稿を非公開にする** | 今「公開中」の投稿をぜんぶ非公開にします |
| **このツールで非公開にしたものを公開する** | このアプリで非公開にした投稿だけを公開に戻します (安全) |
| **非公開のものを全部公開する** | 今「非公開」の投稿をぜんぶ公開にします (もともと非公開だったものも公開されるので注意) |

ボタンを押すと確認ダイアログが出るので、内容を確認して OK を押してください。

進捗バーで何件目を処理しているか見られます。途中で止めたい時は **「中断」** ボタン。

完了するとダイアログでお知らせします。

### 4. リクエスト間隔 (1件ごとの待機時間)

Fantia への負荷を減らすため、投稿 1 件ごとに数秒待つようになっています。デフォルトのままで問題ありません。

## こんなときは

### Q. 「このツールで非公開にしたものを公開する」と「非公開のものを全部公開する」って何が違うの？

- **「このツールで非公開にしたものを公開する」** は、このアプリで非公開にしたものだけを覚えていて、それだけを公開に戻します。 **もともと非公開だった投稿は触りません。** 普通はこっちを使えば OK です。

- **「非公開のものを全部公開する」** は、今 Fantia 上で非公開になっているものを全部公開します。 **もともと非公開にしていた投稿も公開されてしまう** ので、本当に全部出していい時だけ使ってください。

### Q. アプリを一度閉じても、「このツールで非公開にしたもの」の記憶は残る？

残ります。 OS のユーザーデータ領域に履歴を保存しているので、アプリを終了しても次回起動時にちゃんと再公開できます。

### Q. 履歴をバックアップしたい

操作パネルの **「履歴を CSV ダウンロード」** で、非公開にした投稿の ID 一覧を CSV で保存できます。

### Q. 履歴を消したい

**「非公開化履歴をクリア」** で消せます。クリア後はこのアプリから一括公開できなくなるので注意。

### Q. ログインし直したい

右側の Fantia ブラウザでログアウトすれば普通にログアウトできます。 アプリ自体の Cookie 情報はアプリ専用の領域に保存されているので、普段使っているブラウザの Fantia ログイン状態には影響しません。
